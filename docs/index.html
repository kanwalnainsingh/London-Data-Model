MVP Scope

KT19 only Official data only No admissions prediction Local file pipeline

The active MVP is state-funded mainstream school discovery for KT19 with distance, accessibility, Ofsted context, and explicit data quality labeling.

What Works Now

  • Pipeline orchestration and CLI
  • Official and sample input modes
  • CSV schema validation for official files
  • Phase, status, and scope filtering
  • Distance, accessibility, and proximity logic
  • Quality flags and manifest output

Current Blocker

Real school rows are not committed yet because the repo does not contain the required local official source files.

Run It Now

Sample Mode

Run It For Real

Official Mode

PYTHONPATH=src python3 -m london_data_model.cli schools run --area KT19 --input-mode official

This is the command you will use once you place the official local files at the configured paths in data/raw/.

What You Need For The First Real Test

  1. Place the official GIAS establishments file at data/raw/gias_establishments.csv.
  2. Place the official Ofsted state-funded schools file at data/raw/ofsted_state_funded_schools.csv.
  3. Confirm the real file headers match the configured column maps.
  4. Run official mode from the CLI.
  5. Inspect the generated CSV, JSON, summary, and manifest outputs.
