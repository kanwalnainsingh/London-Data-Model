London Boroughs

Pilot Area

KT19 — Starting scope

KT19 was the first area used to test and validate the pipeline. It is a postcode district within the Royal Borough of Kingston upon Thames (LA 314).

Run the Pipeline

Single borough

ldm schools run --area barnet --input-mode official

Runs for any borough with a config in configs/areas/.

All London boroughs

ldm schools run --area london --input-mode official

Loads GIAS data once, filters per borough, publishes 33 output sets.

Borough subset

ldm schools run --area london \
  --boroughs barnet,camden,islington \
  --input-mode official

Comma-separated borough IDs.

KT19 pilot (sample mode)

ldm schools run --area KT19 --input-mode sample

Runs the original pilot area in sample mode.

Required Local Data Files

  1. Place the GIAS establishments CSV at data/raw/gias_establishments.csv
  2. Place the Ofsted state-funded schools CSV at data/raw/ofsted_state_funded_schools.csv
  3. Verify CSV headers match the configured column maps in configs/pipeline/schools.yml
  4. Run official mode — the pipeline filters by LA code per borough automatically
Official data files are not yet placed. All school counts show 0 until real files are provided.
