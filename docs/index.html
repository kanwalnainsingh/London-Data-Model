London Boroughs

Pilot Area

KT19 — Starting scope

KT19 was the first area used to test and validate the pipeline. It is a postcode district within the Royal Borough of Kingston upon Thames (LA 314).

Run the Pipeline

Single borough

ldm schools run --area barnet --input-mode official

Runs for any borough with a config in configs/areas/.

All London boroughs

ldm schools run --area london --input-mode official

Loads GIAS data once, filters per borough, publishes 33 output sets.

Borough subset

ldm schools run --area london \
  --boroughs barnet,camden,islington \
  --input-mode official

Comma-separated borough IDs.

KT19 pilot (sample mode)

ldm schools run --area KT19 --input-mode sample

Runs the original pilot area in sample mode.

Data Sources

Official data is downloaded automatically when you run in official mode — no manual file placement needed.

GIAS — Get Information About Schools

Department for Education establishment register. Downloaded fresh each run from the DfE open-data endpoint.

Ofsted — Five-year inspection data

Latest Ofsted state-funded school inspection results. Auto-resolved from the GOV.UK publications page each run.

Pipeline auto-downloads GIAS and Ofsted data. Run ldm schools run --area KT19 --input-mode official to generate fresh data.
