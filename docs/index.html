London Boroughs

Pilot Area

KT19 — Starting scope

KT19 was the first area used to test and validate the pipeline. It is a postcode district within the Royal Borough of Kingston upon Thames (LA 314).

Run the Pipeline

Single borough

ldm schools run --area barnet --input-mode official

Runs for any borough with a config in configs/areas/.

All London boroughs

ldm schools run --area london --input-mode official

Loads GIAS data once, filters per borough, publishes 33 output sets.

Borough subset

ldm schools run --area london \
  --boroughs barnet,camden,islington \
  --input-mode official

Comma-separated borough IDs.

KT19 pilot (sample mode)

ldm schools run --area KT19 --input-mode sample

Runs the original pilot area in sample mode.

Data Sources & Freshness

Every school record is assembled from four official open-data sources. The table below shows what each source provides, when it was last pulled, how large the file was, and when the underlying data was published.

SourceCategoryPublisher Update frequencyFields used
GIAS — Get Information About Schools Administrative register Department for EducationDaily Name, address, phase, type, coordinates, postcode, LA code
Ofsted — State-funded inspection outcomes Inspection data OfstedMonthly Overall effectiveness grade, inspection date
DfE KS4 — GCSE performance tables Performance statistics Department for EducationAnnual (January) Progress 8, Attainment 8, % grade 4+/5+ English & Maths
DfE KS5 — A-level performance tables Performance statistics Department for EducationAnnual (October/November) Avg point score, % A*/A, pass rate, number of students

Run the pipeline in official mode to populate live fetch timestamps and file sizes.

Pipeline auto-downloads GIAS and Ofsted data. Run ldm schools run --area london --input-mode official to generate fresh data.
