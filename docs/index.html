Run It For Real

Official Mode

PYTHONPATH=src python3 -m london_data_model.cli schools run --area KT19 --input-mode official

This is the command you will use once you place the official local files at the configured paths in data/raw/.

What You Need For The First Real Test

  1. Place the official GIAS establishments file at data/raw/gias_establishments.csv.
  2. Place the official Ofsted state-funded schools file at data/raw/ofsted_state_funded_schools.csv.
  3. Confirm the real file headers match the configured column maps.
  4. Run official mode from the CLI.
  5. Inspect the generated CSV, JSON, summary, and manifest outputs.
